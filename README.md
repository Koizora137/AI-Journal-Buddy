# AI-Journal-Buddy
Simple AI-powered journaling web application that generates reflective responses based on user entries.
Technologies
	•	HTML
	•	CSS
	•	JavaScript
	•	Node.js
	•	OpenAI API

Note: This repository contains only the frontend.
The Node.js backend handling the OpenAI API requests is kept local to protect the API key.
